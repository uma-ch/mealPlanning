Recipe planning app that does the following
1. Allows collecting of recipes from websites where you can save the original url and a local copy of the relevant recipe without the additional fluff
2. For a particular recipe user can select the recipe and have a grocery list be generated with categories such as vegetables, dairy, etc
3. Allows recipes to be tagged with tags. Some sample tags are - dinner, healthy, weeknight, quick
4. Has a calendar where you can pick and add different recipes for days of the week
5. Can be used by atleast 2 users.
